### Requisitos para ejecución:
1. Importar el archivo `gimnasio.sql` en MySQL.
2. El servidor MySQL debe estar en el puerto **3307** (o cambiar `ConexionDB.java`).
3. Incluir el driver `mysql-connector-j` en las librerías referenciadas.

### Funcionalidades:
- Login y Registro dinámico (Swing).
- Gestión de Clientes y Clases (DAO).
- Consulta de Matrículas con JOIN (DTO).
- Transacciones atómicas y borrado en cascada.
